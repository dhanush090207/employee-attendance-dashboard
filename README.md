# Attendance Dashboard

## Overview
This project is a Power BI dashboard that analyzes attendance data.

## Features
- Total enrolled, present, absent KPIs
- Team-wise attendance analysis
- Senior manager-wise performance
- Gender distribution
- Job level insights

## Tools Used
- Power BI
- Data Visualization

## Insights
- Identified teams with highest and lowest attendance
- Compared attendance across managers
- Analyzed gender distribution
